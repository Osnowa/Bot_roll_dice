# 🎲 Roll Dice Telegram Bot

Телеграм-бот на Aiogram 3, в котором пользователи играют в кости против бота, зарабатывают очки и соревнуются в лидерборде.

## Функционал

* Регистрация пользователей
* Игра в кости против бота
* Подсчет очков за победы и ничьи
* Статистика игрока
* Режим игры 3 подряд
* Хранение данных в PostgreSQL
* Кэширование через Redis

## Технологии

* Python 3.13
* Aiogram 3
* PostgreSQL
* SQLAlchemy Async ORM
* Redis
* Docker & Docker Compose
* Pytest
* GitHub Actions (CI)
* Git

## Особенности реализации

* Проект построен по многослойной архитектуре (Handlers → Services → Repository → Database)
* Бизнес-логика вынесена в сервисный слой и не зависит от Telegram API
* Работа с базой данных изолирована через Repository Pattern
* Используется асинхронный стек (Aiogram, SQLAlchemy Async, asyncpg)
* Реализовано разделение ответственности между слоями приложения
* Подключено кэширование через Redis
* Настроены Unit и Integration тесты
* Настроен CI-пайплайн через GitHub Actions
* Проект полностью контейнеризирован с помощью Docker и Docker Compose
* Конфигурация приложения вынесена в переменные окружения (.env)


## Архитектура проекта

Проект разделен на слои:

* Handlers — обработка Telegram-сообщений
* Service Layer — бизнес-логика приложения
* Repository Layer — работа с базой данных
* Database Models — модели SQLAlchemy

## Тестирование

Реализованы:

* Unit-тесты
* Integration-тесты для работы с БД
* Mock и AsyncMock для изоляции зависимостей

Покрыты тестами:

* Игровая логика
* Сервисный слой
* Репозиторий базы данных
* Регистрация пользователей
* Начисление очков и статистики
